New Terms:
    - Hobbs/Tacho/Timestamp
        1. HOBBS TIME
        Definition:
        - Measures real elapsed time when aircraft is powered or engine running.

        How it works:
        - Starts: Engine oil pressure ON or master switch ON
        - Stops: Engine/power OFF

        Measures:
        - Actual clock time (like stopwatch)

        Usage:
        - Primary billing metric
        - Aircraft rental charges

        Key Points:
        - Includes taxi, idle, run-up time
        - Independent of RPM

        2. TACH TIME (TACHOMETER)
        Definition:
        - Measures engine usage based on RPM, not real time.

        How it works:
        - Normal RPM → real-time rate
        - Low RPM → slower time
        - High RPM → faster time

        Measures:
        - Engine wear

        Usage:
        - Maintenance scheduling
        - Oil change, inspections

        Key Points:
        - Usually less than Hobbs time
        - Better for maintenance, not billing

        3. TIMESTAMP
        Definition:
        - Records exact time of flight events.

        Events:
        - Engine start
        - Takeoff
        - Landing
        - Shutdown

        Measures:
        - Exact time points (ISO format)

        Derived Metrics:
        block_time = shutdown - engine_start
        air_time   = landing - takeoff
        taxi_time  = block_time - air_time

        Usage:
        - Flight logs
        - Student records
        - Compliance and analytics


        COMBINED USAGE

        Billing:
        flight_cost = hobbs_time * hourly_rate

        Maintenance:
        if tach_time exceeds threshold → trigger service

        Training / Logs:
        student_air_time = landing - takeoff

        KEY TAKEAWAY

        Hobbs = Billing (money)
        Tach  = Maintenance (engine health)
        Time  = Audit & analytics (truth)
